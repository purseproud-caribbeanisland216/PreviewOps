## 🚀 What PreviewOps Does

PreviewOps helps you spin up a full app preview for each pull request. It is built for teams that want to check changes before they merge code. You can use it to see a live version of your app without hand setup, YAML files, or long waits.

Use it when you want:
- A clean preview for each PR
- Less manual setup
- Faster review before merge
- A simple path from change to live test
